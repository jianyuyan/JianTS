# Jian 社区许可协议（JCL）

**版本 1.0 | 生效日期：2026年7月10日**

---

## 前言

Jian 社区许可协议（Jian Community License，简称 JCL）是一份为 **Jian 体系语言规范**（包括但不限于 Jian、JianPy、JianTS、JianCS）设计的源代码可用许可证。

**核心理念：**

> **个人与开源项目免费使用，商业闭源产品需要付费。**

JCL 旨在平衡以下目标：

1. **鼓励采用与创新**：个人开发者、学术研究、开源项目可自由使用，无需任何费用。
2. **保护生态可持续性**：大型商业组织若将规范用于商业发行版或云服务，需购买授权以支持生态发展。
3. **明确规则与边界**：清晰界定「免费使用」与「商业授权」的边界，消除法律灰色地带。
4. **防止专利碰瓷**：以「现有技术」身份公开规范，阻止第三方就核心语法规则申请专利。


## 第一条：定义

**1.1 「规范」**：指 Jian 体系下各语言的语法规范文档，包括但不限于：
- Jian 语法规范
- JianPy 语法规范
- JianTS 语法规范
- JianCS 语法规范
- 以及后续由版权方发布的任何衍生语言规范文档。

**1.2 「版权方」**：指 Jian 语言规范版权持有人（Jian Language Foundation / 版权声明中指定的个人或实体）。

**1.3 「个人」**：指以个人身份而非代表任何组织使用本规范的个体。

**1.4 「组织」**：指公司、合伙企业、非营利组织、政府机构或其他法律实体。

**1.5 「开源项目」**：指采用 OSI 批准的开源许可证（如 MIT、Apache 2.0、GPL、BSD 等）发布源代码的项目。

**1.6 「商业使用」**：指直接或间接以营利为目的的下列行为：
- 开发、分发、销售基于本规范的编程语言或编译器；
- 将基于本规范的编程语言作为云平台、PaaS、SaaS 服务的一部分对外提供；
- 将本规范嵌入到商业培训课程、教材或认证体系中；
- 其他以收费方式提供与本规范相关的产品或服务。

**1.7 「商业授权」**：指版权方根据本协议第三条授予的、允许商业使用的付费许可。


## 第二条：免费使用条款

以下场景无需申请商业授权，**自动获得免费使用权利**：

**2.1 个人学习与研究**
- 个人开发者学习、理解本规范；
- 个人开发者基于本规范编写实验性代码或原型；
- 学术研究机构将本规范用于非商业性学术研究。

**2.2 开源项目**
- 基于本规范开发的开源编程语言工具；
- 基于本规范开发的开源语法分析器、转换器；
- 基于本规范开发的开源教学工具或示例代码。

**2.3 内部使用（小型组织）**
- 员工总数 ≤ 50 人的组织，可在内部环境中使用本规范；
- 该使用不得向第三方提供产品或服务。

**2.4 衍生规范创作**
- 可基于本规范创作衍生语法规范文档，但须：
  - 在衍生文档中注明「基于 Jian 体系语法规范衍生」；
  - 不得暗示与版权方存在从属关系；
  - 遵循「同样授权条款共享」原则，衍生规范须使用相同许可证。


## 第三条：商业授权条款

**3.1 需要商业授权的场景**

以下场景必须向版权方购买商业授权：

1. **大型组织内部使用**：员工总数 > 50 人的组织，在内部开发、部署基于本规范的工具或系统；
2. **商业发行版语言**：任何组织发布基于本规范的、用于营利目的编程语言或编译器；
3. **云平台服务**：以 SaaS、PaaS 或类似形式对外提供基于本规范的编程环境或编译服务；
4. **商业培训与教材**：将本规范作为付费培训课程、认证考试、付费教材的核心内容；
5. **其他营利性使用**：任何不在第二条免费使用范围内的营利性行为。

**3.2 授权费用**

商业授权费用由版权方与授权方协商确定，通常基于以下因素：
- 使用规模和范围；
- 组织类型与营收规模；
- 商业模式的营利性质。

未经版权方书面许可，任何组织或个人不得使用以下标识：
- 「Jian」、「JianPy」、「JianTS」、「JianCS」；

- 暗示与版权方存在关联、认可或背书的表述。

向本规范贡献内容（包括但不限于提交 Issue、Pull Request、修改建议）即表示贡献者同意：
- 将其贡献内容的知识产权授予版权方；
- 贡献内容将成为本规范的一部分，按照 JCL 许可证发布；
- 贡献者放弃对被贡献内容的「道德权利」主张。

**4.4 「现有技术」声明**

本规范以公开形式发布，构成相关技术领域的「现有技术（Prior Art）」。任何第三方不得就本规范中公开的语法规则、关键词映射关系等核心设计申请专利。若第三方申请并获得了相关专利，版权方保留：
- 以本规范的公开时间戳作为证据，请求宣告该专利无效的权利；

## 第六条：违约与终止

**6.1 自动终止**

任何违反本协议条款的行为将导致本协议授予的所有权利自动终止，无需另行通知。

**6.2 恢复权利**

因违约导致的权利终止后，违约方须：
1. 立即停止所有基于本规范的使用行为；
2. 删除或销毁所有基于本规范的衍生作品；
3. 向版权方书面说明违约情况及整改措施。
版权方有权根据违约方的补救措施酌情恢复其权利。

**6.3 违约金**

对于商业授权范围内的违约行为，版权方有权追索：
- 自违约行为发生之日起的授权费用（按版权方标准费率计算）；
- 因违约行为造成的实际损失（包括但不限于律师费、诉讼费、调查费）。
